Added a rebuild feature to lb

[home—link—standalone] As previously discussed this blog is run using lb by Luke Smith. It is a really nice bit of software but lacks a few features I wish it had. The first of the features being a rebuild function. A rebuild function would rebuild all the HTML files in the blogs folder so that as I change the template the files all change. For my first project with modifying lb, I decided to tackle that problem and that feature has now been added. Going through Github commit history for my fork of lb is rather a big mess due to some issues with exponential duplication of the RSS and HTML feed, but all the changes made were on lines 64 to 95 of the lb executable in commit #1fbbbea.
